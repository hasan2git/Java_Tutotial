Java Setup is a java tutorial site where developers can find detailed example of basic java applications of different java frameworks. For java there are so many apis and frameworks. Javasetup is here to help you to learn how to use those apis and frameworks in a simple way so that one can understand very easily and in much lesser time. The goal is to help you in every step of setting up an java application for first time, so that you will not face any difficulties when you learn different java apis and frameworks. Every exmaple in this tutorial tutorial contains step by step guide with screenshots and source code ready to use.



The basic requirement for setting up any java application is
  • Use an IDE.
  • Create a java project.
  • Add supporting libraries.
  • Write the code.
  • Build the code.
  • Run the application.
Here we cover all these above mentioned points.
