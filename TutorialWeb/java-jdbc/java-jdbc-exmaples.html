Java JDBC Database Access

In This tutorial we will learn how to write a java code to access database using JDBC API.

To access database using JDBC API, we need to follow below 5 steps :

  • Establishing a connection.
  • Create a statement.
  • Execute the query.
  • Process the ResultSet object.
  • Close the connection.

JDBC Architecture



There are 2 ways to establish a JDBC connection

  1. Using DriverManager class.
  2. Using Datasource interface

Examples are given for both the ways of establising a JDBC connection with the CRUD(Create,Read,Delete,Update) operations.
